Use pipes to transform strings, currency amounts, dates, and other data for display. Pipes are simple functions you can use in template expressions to accept an input value and return a transformed value.

Number

  • {{ pi }} - start value
  • {{ pi | number}} - e | number
  • {{ pi | number:'1.0-5'}} - e | number:'1.0-5'
  • {{ pi | number:'10.0-5'}} - e | number:'1.0-5'
  • {{ pi | number: '0.0-0'}} - e | number: '0.0-0'
  • {{ pi | currency }} - pi | currency
  • {{ pi | percent }} - pi | percent
  • {{ pi | percent:'3.0-2' }} - pi | percent:'3.0-2'

Async

  • {{promise | async}} - promise
  • {{ dateObservable | async | date:'HH:mm:ss'}} - observable

String

  • {{ str }} - start value
  • {{ str | uppercase }} - str | uppercase
  • {{ str | lowercase }} - str | lowercase
  • {{ str | lowercase | uppercase }} - str | lowercase | uppercase
  • {{ str | titlecase }} - str | titlecase
  • {{ str | titlecase | slice:1:5 }} - str | titlecase | slice:1:5

Date

  • {{ date }} - start value
  • {{ date | date }} - date | date
  • {{ date | date: 'short' }} - date | date: 'short'
  • {{ date | date: 'fullTime' }} - date | date: 'fullTime'
  • {{ date | date: 'dd MMM y, EEEE' }} - date | date: 'dd MMM y, EEEE'

Obj 

  • {{ obj | json }} - obj | json

Custom pipe - multBy

  • {{ pi }} - start value
  • {{ pi | multBy:2 }} - pi | multBy:2
  • {{ pi | multBy:3 }} - pi | multBy:3
  • {{ pi | multBy:6 | number:'2.0-3'}} - pi | multBy:6 | number:'2.0-3'
  • {{ pi | multBy:20 }} - pi | multBy:20
